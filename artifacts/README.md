# Artifacts

Bu klasor, uretilmis ve inceleme amacli artefact dosyalarini toplar.

## Yapı

- `artifacts/objects`
  - derleme sonucu olusan `.o` dosyalari

- `artifacts/audio/reference`
  - reference tarafinda uretilmis WAV dosyalari

- `artifacts/audio/compare`
  - reference/fixed karsilastirma kosularinda uretilmis WAV dosyalari

## Not

- `test_wav` ayri tutulur
- oradaki dosyalar test veri seti ve bu veri setine ait batch output'lari icindir
